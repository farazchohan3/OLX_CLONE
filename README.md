### OLX Clone Project

This project is a clone of the popular classified ads website, OLX. It allows users to view product listings on the homepage and view the details of individual products on a separate page.More Feature will be add occasionally

### Features

View a list of products on the homepage
View the details of individual products on a separate page

### Future Development

We plan to add the following features to this project:

1. User registration and login
2. User profile management
3. Ad posting and management
4. Ad search and filtering
5. Contact form for messaging sellers

### Technologies Used

1. HTML/CSS/JavaScript/bootstrap for frontend development
2. React for frontend UI components
3. React Router v6 for page navigation
4. Redux Toolkit for state management

### Getting Started

To get started with this project, follow these steps:

1. Clone the repository to your local machine
2. Install dependencies with npm install
3. Start the development server with npm start
